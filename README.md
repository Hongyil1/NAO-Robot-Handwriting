![test1](https://user-images.githubusercontent.com/22671087/40587069-e0259240-620d-11e8-9812-7054dd4b8272.jpg)
## Prerequisites
* Python2.7
* NAO Robot
* Wacom Intuos (CTH-480) pen-tablet
* pynaoqi
* pygame
* Tkinter

## How to use
```
python Main.py -ip your_ip -port your_port
```
